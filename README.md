## Tugas 4 - Struktur Data Python
Python menyediakan 4 struktur data utama yang beda beda karakteristiknya sesuai dengan kebutuhan, yaitu List, Tuple, Set, dan Dictionary

List cocok untuk data yang perlu dimanipulasi (tambah, hapus, dan sebagainya)
Tuple cocok untuk data yang tetap, hanya bisa di unpacking
set digunakan dalam operasi himpunan dan menjamin keunikan
dictionary menyimpan data berbasis key dan value

Terdapat 6 cara untuk memanipulasi data dari list=
- Slicing Data
- Append Data
- insert Data
- Extend Data
- Pop Data
- Remove Data
